# Basic Intrusion Detection System (IDS)

## Description
This project simulates a basic Intrusion Detection System by monitoring repeated IP address attempts and generating alerts for suspicious activity.

## Features
- Records IP address attempts
- Tracks repeated access attempts
- Generates alerts for suspicious activity
- Simple simulation of IDS behavior

## Language Used
Python

## Files Included
- main.py → IDS program
- README.md → Project documentation

Run the program:

```bash
python main.py
