# Secure Coding Review

## Description
This project performs a basic security review on Python files and detects common insecure coding practices.

## Features
- Detects usage of eval()
- Detects usage of exec()
- Identifies hardcoded passwords
- Detects possible SQL Injection patterns

## Language Used
Python

## Files Included
- main.py → Security review program
- test.py → Sample file for testing
- README.md → Project documentation

Run the program:

```bash
python main.py
